## Get url of current repo

`git remote get-url origin`

also command `openreposite` on my personal machine will open the page in the default browser

## Set url of current repo

first time:
`git remote add origin https://www.github.com/USERNAME/REPOSITORY.git`

subsequent times:
`git remote set-url origin https://github.com/USERNAME/REPOSITORY.git`

## Check credentials

`git config -l`

## Store credentials

`git config credential.helper store`

## "Logout"

`git config --global --unset-all`
